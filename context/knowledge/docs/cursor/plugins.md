# Cursor Plugins

Source: https://cursor.com/docs/plugins

## Overview

Plugins extend Cursor with additional tools and integrations.

Examples:

* GitHub automation
* Slack notifications
* Figma integration
* CI/CD pipelines

## Installing Plugins

Plugins can be installed through:

```
/add-plugin
```

## Plugin Capabilities

Plugins can:

* expose tools
* integrate APIs
* automate workflows

## Example Plugin

```
GitHub Plugin

Capabilities:
- create pull requests
- run workflows
- comment on issues
```

## Use Cases

* DevOps automation
* code review bots
* project management integration
